# Azuki
Azuki is a simple Map evaluation tool

# Supported Expressions

* Evaluation Expressions
    * equals
    * equalsIgnoreCase
    * contains
    * containsIgnoreCase
    * startsWith
    * startsWithIgnoreCase
    * endsWith
    * endsWithIgnoreCase
    * pattern
* Combination Expressions
    * all
    * any
    * none
* Wrapper Expressions
    * not
